# KeyLogger Overview
This is a simpole key logger implemented in Python, it captures keystrokes and saves them to a file. This was made for educational purposes only. Use it responsibly and ethically. Don't be an idiot, im not responsible for your actions xoxo.
## Features
- Captures keystrokes
- Saves keystrokes to a file
- Runs in the background
- Cross-platform support (Windows, macOS, Linux)

## Requirements
- Python 3.x
- pynput library
- os library
- datetime library
- pathlib library

## Installation
clone the repo
install pynput library using pip

**For MacOS (lmao)
install pyobjc-framework-AppKit using pip

**For Linux
install xdotool 
install python-xlib using pip
